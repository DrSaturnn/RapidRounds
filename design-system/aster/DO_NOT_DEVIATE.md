# Do Not Deviate

The Aster concept sheets are specifications, not inspiration.

Never:

- enlarge the eyes beyond the canonical proportions
- shrink the visor
- change the helmet silhouette
- simplify Aster into a generic round robot
- remove ceramic seams
- remove the chest crystal
- add a mouth
- add eyebrows
- add human facial features
- add weapons or aggressive expressions
- make Aster look childish
- make Aster look like a chatbot avatar
- place Aster where he competes with medical reasoning

If fidelity conflicts with implementation convenience, prefer fidelity.

If Aster would distract from clinical reasoning, reduce scale or hide him.
