# Aster Animations

This folder is reserved for animation definitions and approved animation exports
from the future rigged Aster model.

Animations must be identity-preserving and subtle:

- idle hover
- blink
- thinking tilt
- happy glow
- celebrating sparkle
- sleepy rest
- reading map
- teaching
- exploring

Runtime may use light CSS motion on approved still assets, but it must not
reconstruct or redraw Aster.
