# Orthographic Model Sheet

This folder is reserved for approved orthographic renders from the future single
rigged Aster Blender model.

Required files:

- `Front.png`
- `ThreeQuarterFront.png`
- `Side.png`
- `ThreeQuarterBack.png`
- `Back.png`
- `Top.png`

Do not place generated reinterpretations here. These views must be rendered from
the canonical model calibrated against `../neutral.png`.
