# Discoveries

## 2026-06-27

### Learning

- Unknown is a different cognitive state than incorrect.
- Learners frequently stop one node early in the reasoning chain.
- Preserve cognitive momentum.
- Mechanism-first explanations are remembered better than definition-first explanations.
- Pertinent negatives function as positive evidence.

### Product

- "Your answer becomes correct when..." is more educational than simply showing the correct answer.
- Immediate and ultimate management should be distinguished.
- Semantic matching should recognize complete clinical propositions.
