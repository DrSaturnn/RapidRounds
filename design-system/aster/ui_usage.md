# Aster UI Usage

Aster supports the interface. He is never the interface.

Use Aster for:

- expedition progress
- overworld route markers
- session companion panels
- milestone celebrations
- calm loading or resting states

Avoid Aster in:

- dense answer explanations
- medical decision tables
- grading surfaces
- dashboards or analytics panels
- places where he would obscure the answer input or teaching flow

Scale guidance:

- `tiny`: 16-24px map sprite
- `small`: 56-80px companion marker
- `medium`: 100-140px companion panel feature

At small sizes, preserve silhouette, visor, eyes, and chest crystal first.
